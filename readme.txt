Display your RokGallery galleries as an slideshow in the background of your site.

== Description ==

This plugin enables you to display your RokGallery galleries as an slideshow in the background of your site. Requires the <a href="http://www.rockettheme.com/">RokGallery</a> plugin for WordPress.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -> Widgets and add the widget.
4. Enjoy!
